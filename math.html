Предел последовательности

Число a называется пределом последовательности {An} (n-натуральное число) при n стремящемся к бесконечности Если для всякого Е(эпсилон) > 0 существует номер n0(n0-натуральное число) такой, что для всякого n>n0 выполняется условие |An-a|< E


Предел функции

Число A называют пределом функции в точке x0(или при x -> x0), если для любого положительного E(эпсилон) найдётся такое положительное число δ(дельта), что для всех x из δ-окрестности точки x0 справедливо неравенство: |f(x)-A| < E

Теоремы Вейерштрасса:

#Первая:

Если функция u=f(M) непрерывна на замкнутом ограниченном множестве {M}, то она ограничена на этом множестве


#Вторая:

Если функция u=f(M) непрерывна на замкнутом ограниченном множестве {M}, то она достигает на этом множестве своих точных верхней и нижней граней.


#Третья(Теорема Больцано-Вейерштрасса):

Из любой ограниченной последовательности {Mn} точек m-мерного евклидова пространства можно выделить сходящуюся подпоследовательность


Евклидово пространство - это пространство свойства которого описываются аксиомами евклидовой геометрии


Теорема Ролля:

Пусть функция f(x) непрерывна на сегменте [a,b] и дифференцируема во всех внутренних точках этого сегмента. Пусть, кроме того, f(a)=f(b). Тогда внутри сегмента[a,b] найдётся точка E(эпсилон) такая, что значение производной в этой точке равно нулю


Теорема Лагранжа:

Теорема Ферма

Если функция определена в некоторой окрестности точки, принимает в этой точке наибольшее(наименьшее) значение и имеет конечную или определённого знака бесконечную производную, то эта производная равна 0.


Бесконечно малые


Бесконечно малая функция это функция стремящаяся к 0
Сравнение бесконечно малых функций

  • то функция α(x) является б.м. более высокого порядка малости, чем β(x) при x -> a


  • то функция α(x) является б.м. более низкого порядка малости, чем β(x) при x -> a


  • то функции α(x) и β(x) называются б.м. одного порядка малости при x -> a


  • то функции α(x) и β(x) называются эквивалентными при x -> a


  • то функции α(x) и β(x) называются несравнимыми б.м. при x -> a

Схема исследования функции


1) Область определения функции


2) Особенности функции (чётность, нечётность, переодичность)


3) Точки пересечения с осями, интервалы знакопостоянства.


4) Точки разрыва, характер точек разрыва.

6) Исследование с помощью второй производной (интервалы выпуклости,вогнутости, точки перегиба).


7) Наклонные ассимптоты
